# 🎥 Como Gravar a Demonstração

## Método Mais Simples: Windows Game Bar

1. **Abra a simulação** no navegador (já deve estar aberta)
   
2. **Pressione**: `Win + G`
   - Isso abre o Xbox Game Bar

3. **Clique no botão de gravar** (círculo vermelho)
   - Ou pressione: `Win + Alt + R` para começar

4. **Deixe rodar por 30-60 segundos**
   - Mostre os carros aprendendo
   - Mostre as estatísticas atualizando
   - Mostre a rede neural piscando

5. **Pare a gravação**: `Win + Alt + R` novamente

6. **Vídeo salvo em**: 
   ```
   C:\Users\mathe\Videos\Captures\
   ```

---

## O que Mostrar na Gravação

✅ **Painel de Controle** (direita) - estatísticas ao vivo
✅ **100 carros azuis** aprendendo simultaneamente  
✅ **Tráfego vermelho** aleatório
✅ **Rede neural** (esquerda) com neurônios ativos
✅ **Colisões** - carros ficando cinzas
✅ **Melhor carro** - azul sólido com sensores visíveis

---

## Dica Pro
Clique em **Reset Generation** algumas vezes antes de gravar para ter uma geração mais avançada navegando melhor!
